The Crowdfund blueprint serves as a base implementation for the following use cases:

  - Participating in activities by paying with tokens of a different chain;

**Contracts:**
  - [Tezos - IBCF_Crowdfunding.py](https://github.com/Acurast/acurast-hyperdrive/blob/main/contracts/tezos/blueprints/IBCF_Crowdfunding.py)
  - [EVM - IBCF_Crowdfunding.sol](https://github.com/Acurast/acurast-hyperdrive/blob/main/contracts/evm/blueprints/IBCF_Crowdfunding.sol)

### Using the demo

1. First you need to have some Ether on Goerli testnet. You can get some from a [faucet](https://goerli-faucet.pk910.de/);

2. Once you have some Ether, you can call the EVM [Crowdfunding](https://goerli.etherscan.io/address/0x79a20860c063a70EB483f5b6fb70b9CaDb022729) contract as shown below:
