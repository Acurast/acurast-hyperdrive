## Views

### `verify_proof`

**Signature:**

```
function verify_proof(
    uint snapshot_number,
    bytes memory owner,
    bytes memory key,
    bytes memory value,
    bytes32[2][] memory proof
);
```

**Output type:** `void`

**Description:**
Contracts in a EVM chain can call this method to validate that a given state exists on the Tezos blockchain at a snapshot.

It first asserts that the snapshot state was signed by at least `X` validators and then validates the proof against the snapshot state.
